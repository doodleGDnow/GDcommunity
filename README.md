# 🎮 GDCommunity

GDCommunity is a community platform for Geometry Dash players. Chat with other members, discover new levels, discuss updates, share achievements, and connect with the global Geometry Dash community.

## ✨ Features

* 💬 Global community chat
* 👀 Read messages without an account
* 🔒 Authentication required to send messages
* 👤 User profiles
* 😀 Emoji support
* 🛡️ Secure login and registration
* 📱 Responsive design
* 🎮 Geometry Dash inspired interface

## 🚀 Planned Features

* Google reCAPTCHA
* Private messaging
* User roles and permissions
* Community events
* Level sharing system
* Leaderboards
* Notifications

## 🛠️ Technologies

* HTML
* CSS
* JavaScript
* Firebase Authentication
* Firebase Firestore
